## ✨ Novas Funcionalidades

### 1. 📋 Kanban Board
**Onde**: Work Items → Kanban View  
**O que faz**: Arraste work items entre colunas para mudar status

### 2. 🎯 Navegação por Steps
**Onde**: Barra de progresso em qualquer projeto  
**O que faz**: Clique nos steps para navegar rapidamente

### 3. 📥 Export Markdown
**Onde**: Botão em qualquer documento  
**O que faz**: Exporta documento para .md em 1 clique

### 4. 📊 AI Statistics
**Onde**: Menu → AI Stats  
**O que faz**: Mostra uso e custos de IA em tempo real

## 🐛 Problemas?

### Frontend não compila
```bash
cd frontend
rm -rf node_modules
npm install
```

### Backend não inicia
```bash
docker-compose restart postgres
uv run python scripts/migrate_ai_stats.py
```

### Kanban não funciona
```bash
cd frontend
npm install @dnd-kit/core @dnd-kit/sortable @dnd-kit/utilities
```
